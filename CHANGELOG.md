## [2026-04-06c] — District ROI leaderboard homepage component

### Added
- `components/pmt/DistrictROILeaderboard.tsx` — server component rendering top 10
  congressional districts by federal ROI ratio. Dark header bar, grid-layout rows
  (full-row links to `/districts/[slug]`), IBM Plex Mono data columns, source attribution.
  `overflow-x: auto` for narrow viewport compatibility.
- `public/data/district_roi_top10.json` — top 10 districts by ROI (FY2021–FY2026).
  Generated by `pmte/scripts/export_district_roi_rankings.py`. Added to `.gitignore` whitelist.
- `public/data/district_roi_all.json` — all 429 ranked districts. Same source script.
  Added to `.gitignore` whitelist. Intended for future `/districts` leaderboard page.

### Changed
- `app/page.tsx` — `DistrictROILeaderboard` added between ZIP lookup and Latest Monday
  Reports, bracketed by `1px solid #d4c5a9` horizontal rules.
- `.gitignore` — added `!public/data/district_roi_top10.json` and
  `!public/data/district_roi_all.json` exceptions to the `public/data/` ignore block.

### SITE_ARCHITECTURE changes
- Component Registry — `DistrictROILeaderboard.tsx` row added (Server, homepage)
- Key Data Files — `district_roi_top10.json` and `district_roi_all.json` rows added

### PMTE pipeline (not website code — recorded here for cross-reference)
- `pmte/scripts/export_district_roi_rankings.py` — new script; queries
  `raw_usaspending_district_totals` (FY2021–2026), `raw_irs_soi`, `raw_census_acs`,
  and `politicians` (House, active=1); joins on `(state_abbrev, int(district))` key
  to avoid zero-padding mismatches; writes both JSON files to `website/public/data/`
  and `website/data/`.
- Politicians table audit — full cross-reference against `legislators-current.yaml`.
  Fixed 4 active=0 bugs (Walberg, Thune, Warner, Welch). Inserted 3 new members with
  bioguide collisions resolved (Ashley Moody FL-Senate M001244, Matt Van Epps TN-7
  V000139, Christian Menefee TX-18 M001245). 3 confirmed genuine vacancies remain
  (CA-1, NJ-11, GA-14).
- `raw_usaspending_district_totals` — FY2021–2026 backfill complete (all/contracts/grants,
  441 districts × 6 years × 3 award types).
- District ROI module (`pmte/analysis/district_roi.py`) — Option 0 spend path added
  querying `raw_usaspending_district_totals` as primary source; state_comparison
  deduplication fix (MAX(year) filter on `raw_census_acs`); fallback year bounds fix.
- DRI HTML report — removed duplicate LLM narrative from `_to_html()`; added committees
  to rep block; fixed per-capita formatting (`$5.6K` not `$6K`); removed Top Federal
  Recipients section from HTML, DOCX prompt, and `_build_prompt()`.
